# Arusha Theme — README

Custom WordPress theme for Arusha Group of Company
Built from scratch — no page builder required.

## Files Included
- style.css          → Main styles + theme declaration
- functions.php      → Theme setup, menus, scripts
- header.php         → Navigation + top bar
- footer.php         → Footer columns + bottom bar
- index.php          → Blog listing page
- front-page.php     → Homepage
- single.php         → Single blog post
- page-about.php     → About Us page template
- page-products.php  → Products page (with filter buttons)
- page-contact.php   → Contact page with form
- assets/js/main.js  → Mobile menu + animations

## How to Install

1. Zip the entire `arusha-theme` folder
2. Go to WordPress Dashboard → Appearance → Themes → Add New → Upload Theme
3. Upload the zip file and click Activate

## After Activation

### Set Page Templates:
- Go to each page → Edit → Page Attributes (right sidebar)
- Set template:
  - Home page    → "Front Page" (set as Front Page in Settings → Reading)
  - About Us     → "About Us"
  - Products     → "Products"
  - Contact      → "Contact"

### Set Navigation Menu:
- Appearance → Menus → Create menu → Assign to "Primary Menu"

### Add Products:
- In dashboard, you'll see a new "Products" menu item
- Add products with categories: food, care, home, health
